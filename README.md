# FileServer

Go 언어로 구현한 가벼운 개인 클라우드 파일 서버입니다.
리눅스 서버를 기반으로 동작하며, 같은 네트워크 또는 VPN을 통해 다른 기기에서 파일을 업로드/다운로드할 수 있습니다.

---

## 주요 기능

- 파일 업로드 / 다운로드
- 사용자 로그인 및 계정별 권한 분리
- 파일 공유 링크 생성 (만료 시간 설정)
- 로컬 네트워크 또는 WireGuard VPN을 통한 외부 접속
- 허용된 확장자만 업로드 가능 (보안)

---

## 기술 스택

| 영역 | 사용 기술 |
|---|---|
| 백엔드 | Go (표준 라이브러리 중심) |
| 프론트엔드 | HTML, CSS, JavaScript |
| 데이터베이스 | SQLite (`github.com/mattn/go-sqlite3`) |
| VPN | WireGuard |

> 외부 프레임워크 없이 Go 표준 라이브러리 기반으로 구현하여 의존성을 최소화했습니다.
> DB 드라이버만 최소한으로 사용합니다.

---

## 프로젝트 구조

```
fileserver/
├── main.go
├── go.mod
├── handler/
│   ├── file.go       # 파일 업로드/다운로드 핸들러
│   ├── user.go       # 로그인/회원가입 핸들러
│   └── share.go      # 공유 링크 핸들러
├── model/
│   └── model.go      # 구조체 정의 (User, File, ShareLink, Session)
├── db/
│   └── db.go         # DB 연결 및 쿼리
└── uploads/          # 업로드된 파일 저장 경로
```

---

## 데이터 모델

```
User ──── File      (1:N, 한 유저가 여러 파일 소유)
File ──── ShareLink (1:N, 한 파일에 여러 공유 링크)
User ──── Session   (1:N, 로그인 세션 관리)
```

---

## 접속 방법

### 로컬 네트워크
```
http://<서버 IP>:8080 
```

### 외부 접속 (WireGuard VPN)
WireGuard VPN 연결 후 동일하게 접속 가능합니다.
- 리눅스 서버: 명령어 한 줄로 설치
- 타 기기: WireGuard 앱 설치 후 연결
- 외부 접속 시 DDNS(DuckDNS)를 통해 고정 도메인으로 접속 가능

## 개발 환경

- OS: Linux (Ubuntu)
- Go 버전: 1.21 이상
- DB: SQLite3

---

## 실행 방법

```bash
# 의존성 설치
go mod tidy

# 실행
go run main.go

# 빌드 (단일 실행파일)
go build -o fileserver main.go
./fileserver
```

## 개발 배경

기존 클라우드 서비스(구글 드라이브, Dropbox 등)는 사용 비용이 발생하고 외부 인프라에 의존해야 합니다.
이 프로젝트는 HDD 초기 비용만으로 운용 가능한 자체 클라우드 서버를 목표로 하며,
불필요한 기능을 제거하고 가볍게 설계하여 실사용 불편함을 줄이고자 했습니다.
